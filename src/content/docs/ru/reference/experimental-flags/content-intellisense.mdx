<p>

**Тип:** `boolean`<br />
**По умолчанию:** `false`<br />
<Since v="4.14.0" />
</p>

Включает функции Intellisense (например, автодополнение кода, быстрые подсказки) для записей ваших коллекций контента в совместимых редакторах.

При включении эта функция будет генерировать и добавлять JSON-схемы в директорию `.astro` в вашем проекте. Эти файлы могут использоваться языковым сервером Astro для обеспечения Intellisense внутри файлов контента (`.md`, `.mdx`, `.mdoc`).

```js
{
  experimental: {
    contentIntellisense: true,
  },
}
```

Чтобы использовать эту функцию с расширением Astro для VS Code, вы также должны включить опцию `astro.content-intellisense` в настройках VS Code. Для редакторов, использующих языковой сервер Astro напрямую, передайте параметр инициализации `contentIntellisense: true`, чтобы включить эту функцию.
