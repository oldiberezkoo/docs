Мы приветствуем любой вклад и участников любого уровня подготовки. Как проект с открытым исходным кодом, мы верим в поддержку наших контрибьюторов. Мы рады помочь советами по PR, написанию технической документации и воплощению любых идей в реальность.

:::tip[Открытые проблемы]
Просмотрите [все открытые проблемы (issues) Astro на GitHub с метками `help wanted` и `good first issues`](https://github.com/search?q=org%3Awithastro+label%3A%22good+first+issue%22%2C%22help+wanted%22%2C+state%3Aopen&type=issues) для всех репозиториев кода: код Astro, документация, Starlight, веб-сайты Astro, языковые инструменты, GitHub Actions, боты, компилятор... есть много мест, куда можно внести свой вклад!
:::

Хотите принять более активное участие? Ознакомьтесь с нашим [документом об управлении (Governance doc)](https://github.com/withastro/.github/blob/main/GOVERNANCE.md) для детального описания различных ролей, процессов выдвижения мейнтейнеров, процессов код-ревью и соблюдения Кодекса поведения.

## Способы внести вклад

### Репозитории проекта

Есть много способов внести вклад в проект Astro! У каждого репозитория Astro есть README со ссылкой на файл `CONTRIBUTING.md` в корне проекта.

Посетите [профиль Astro на GitHub](https://github.com/withastro), чтобы найти репозитории для:

- [Основной кодовой базы Astro](https://github.com/withastro/astro), включая официальные интеграции и стартовые шаблоны.

- [Документации Astro](https://github.com/withastro/docs) — целый веб-сайт Astro! Вносите вклад не только в текстовый контент, но и в код Astro, решая задачи доступности (a11y), CSS, UI и UX. Мы также переводим нашу документацию на несколько языков, поэтому нам нужна помощь в переводе всего сайта.

- [Компилятора Astro](https://github.com/withastro/compiler), написанного на Go и распространяемого как WASM.

- [Языковых инструментов](https://github.com/withastro/language-tools) Astro — инструментов редактора, необходимых для языка Astro (файлы `.astro`).

- [Starlight](https://github.com/withastro/starlight) — официального фреймворка документации Astro.

- [Дорожной карты (Roadmap) Astro](https://github.com/withastro/roadmap), где формируется будущее Astro! Идеи, предложения и официальные RFC для проекта Astro.

### Типы вкладов

Помимо написания собственного кода или контента, вы можете внести огромный вклад, участвуя в обсуждениях: оставляя комментарии при проверке PR (пулл-реквестов), добавляя идеи в существующие проблемы (Issues) и обсуждения на GitHub, а также участвуя в наших «закрепленных» задачах по поддержке!

Каждый PR, особенно PR с переводами, нуждается в ревьюерах! Проверка PR и оставление комментариев, предложений или одобряющего «LGTM!» («Looks Good To Me!» / «Мне нравится!») — отличный способ начать работу в любом репозитории и узнать больше об Astro.

У нас также очень активное сообщество в [Discord](https://astro.build/chat)! Мы ценим вклад тех, кто приветствует новых участников, отвечает на вопросы поддержки и делится тем, что они создали с помощью и для Astro! Помимо традиционного вклада через GitHub, Astro признает и поддерживает членов сообщества, которые взаимодействуют с нами, рассказывают об Astro в блогах, видео и докладах на конференциях, а также помогают поддерживать здоровье нашего сообщества.

## Вклад в документацию

У нас есть несколько руководств, которые помогут вам внести вклад в документацию Astro.

Будь то ваш самый первый вклад в open-source, или вам нужно добавить документацию для новой функции Astro, которую вы только что создали, или вы опытный переводчик, ищущий следующую страницу для перевода, или вы хотите узнать больше о помощи в качестве ревьюера PR... у нас есть всё необходимое!

Пожалуйста, посетите наш специальный сайт [Документация *Astro Docs*](https://contribute.docs.astro.build), где вы найдете инструкции, которые помогут вам внести вклад в Astro Docs в качестве корректора, автора, переводчика, создателя функций и даже ревьюера PR.
