Хотите начать изучение Astro с курса или учебного пособия?

Вы можете изучить основы Astro с помощью нашего [официального учебного пособия по созданию блога](/ru/tutorial/0-introduction/), или изучить нашу коллекцию рекомендуемых образовательных материалов по Astro.

:::
