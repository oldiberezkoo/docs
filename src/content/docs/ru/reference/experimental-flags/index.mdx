title: Настройка экспериментальных флагов

Экспериментальные функции доступны только после включения соответствующего флага в файле конфигурации Astro.

```js title="astro.config.mjs"
import { defineConfig } from 'astro/config';

export default defineConfig({
    experimental: {
        // включите экспериментальные флаги,
        // чтобы попробовать новые функции
    },
});
```

Astro предлагает экспериментальные флаги, чтобы предоставить пользователям ранний доступ к новым функциям для тестирования и обратной связи.

Эти флаги позволяют вам участвовать в разработке функций, сообщая о проблемах и делясь своим мнением. Эти функции не гарантируют стабильность и могут включать критические изменения даже в небольших `patch` релизах, пока функция находится в активной разработке.

Мы рекомендуем часто [обновлять Astro](/ru/upgrade-astro/#обновление-до-последней-версии) и следить за примечаниями к выпуску в [журнале изменений Astro](https://github.com/withastro/astro/blob/main/packages/astro/CHANGELOG.md), который проинформирует вас о любых изменениях, необходимых в коде вашего проекта. Документация по экспериментальным функциям всегда будет обновляться только для текущей выпущенной версии.
