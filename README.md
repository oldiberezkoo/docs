## Мы — Astro

Astro — это универсальный веб-фреймворк для создания быстрых сайтов, ориентированных на контент.
Мы хотим, чтобы каждый мог успешно создавать сайты, и для этого мы помогаем всем понять, как работает Astro.


## Статус перевода

Текущий перевод охватывает примерно 90% фундаментальной части фреймворка. Все материалы, выходящие за рамки базовой логики фреймворка, можно найти в официальной документации. Я ожидаю релиза версии v6.

Для сохранения актуальности и предотвращения быстрого устаревания информации не рассматриваются и не будут рассматриваться следующие темы:
- Миграции
- Способы деплоя
- Спонсоры и упоминания их на зеркале
